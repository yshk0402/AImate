# Landing Page Playbook

## Route Convention
- `/{locale}/lp/{campaign}`

## Recommended Structure
1. Value proposition
2. Problem framing
3. Solution overview
4. Social proof
5. CTA block
6. FAQ

## Metadata Checklist
- Title and description are specific to campaign intent
- OG image path is valid in `public/images`
- Internal links point to existing routes

## Iteration Workflow
- Start with `draft`
- Share preview on Vercel preview deployment
- Update copy and section order based on feedback
- Flip to `published` when ready
